# Okka

A toy terminal-based editor written in [Roc](https://www.roc-lang.org).

The editor is based on the online booklet [Build Your Own Text Editor](https://viewsourcecode.org/snaptoken/kilo).

I created a small tui package called rocterm (please give me a better name :D), which the editor uses. It is based on the rust crate [termion](https://crates.io/crates/termion).
